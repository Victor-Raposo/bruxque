# bruxque

---

## Conceitos

A proposta é criar um quiz infantil, com a temática de brusque. As perguntas abordam questões históricas, culturais e sociais da cidade.

## Funcionamento

O jogo é dividido em 3 mundos. Um dos mundos é fácil, outro médio, e outro díficil. Cada um dos mundos contém 5 fases. Além disso, o jogo contém um sistema de vidas, onde a cada mundo, o jogador terá 3 vidas. A cada resposta errada, o jogador perde uma delas, e ao chegar a zero, ele volta ao menu principal, podendo tentar novamente do início.
